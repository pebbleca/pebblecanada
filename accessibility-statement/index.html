Accessibility Statement

Our commitment to making pebblecanada.ca accessible to everyone.

Last reviewed: March 2026

Pebble Canada (Pebble CA Inc.) is committed to ensuring that pebblecanada.ca is accessible to all users, including those who rely on assistive technologies such as screen readers, keyboard navigation, or voice control software. We have designed and built this website in alignment with the Government of Canada Standard on Web Accessibility and the Web Content Accessibility Guidelines (WCAG) 2.0, Level AA.

What We Have Done

The following measures have been implemented across all pages of this website:

  • › Skip navigation link — A "Skip to main content" link is present on every page, allowing keyboard users to bypass the navigation menu and reach the main content directly.
  • › Semantic HTML structure — Pages use correct landmark roles (banner, navigation, main, contentinfo) and a logical heading hierarchy starting with a single <h1> per page.
  • › Visible focus indicators — Keyboard focus is clearly visible on all interactive elements, including links, buttons, and form fields, using a high-contrast purple outline consistent with the site's colour palette.
  • › Descriptive link text and image alt text — All links have meaningful labels and all images carry descriptive alternative text. Icon-only links (such as social media icons) include aria-label attributes.
  • › Accessible forms — All contact form fields are associated with visually-hidden labels using <label for> associations, ensuring screen readers can identify each field correctly.
  • › Bilingual content — The website is available in both English and French, with the language toggle accessible to keyboard users and the lang attribute updated dynamically when the language changes.
  • › Valid HTML — Pages use valid, standards-compliant HTML5 markup without deprecated structural patterns.
  • › Minimum touch targets — Interactive elements, including social icon links, meet a minimum 44×44 pixel touch target size.
  • › Colour contrast — Text and interactive elements meet or exceed the WCAG 2.0 AA minimum contrast ratio of 4.5:1 against their backgrounds.

Known Limitations

We are committed to continual improvement. The following areas are under review:

  • › The full-screen home page slider relies on a third-party JavaScript library. We monitor this component and will improve its keyboard and screen reader behaviour as the library is updated.
  • › Some embedded inline styles inherited from the base theme may not fully meet enhanced contrast targets in all contexts. We are addressing these progressively.
